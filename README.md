# Getting started

## For Windows
| Step         | Command                                                |
|:-------------|:-------------------------------------------------------|
| **git**      | `git clone https://github.com/LaGipeR/Determinant/`    |
| **compile**  | `g++ -std=c++17 -O2 main.cpp >a.exe` | 
| **run**      | `a.exe`   		                |

## For Linux
| Step             | Command                                             |
|:-----------------|:----------------------------------------------------|
| **git**          | `git clone https://github.com/LaGipeR/Determinant/` |
| **switchbranch** | `git checkout linux`                                |
| **compile**      | `g++ -std=c++17 -O2 main.cpp`   		         |
| **run**          | `./a.out`           				 |
